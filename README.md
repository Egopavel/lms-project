# E-Learning System (LMS)

Django-based e-learning platform for managing courses, students, and teachers.

## Features
- User authentication (students & teachers)
- Course management
- Lecture viewing
- Online testing with results tracking
- Student progress monitoring

## Installation
1. git clone https://github.com/Egopavel/lms-project.git
2. cd lms_project
3. python -m venv venv && source venv/bin/activate
4. pip install -r requirements.txt
5. python manage.py migrate
6. python manage.py runserver

## Tech Stack
Python, Django, PostgreSQL/SQLite, HTML/CSS
